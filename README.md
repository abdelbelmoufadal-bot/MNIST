# AI Digits Classification Lab

This project is an interactive Machine Learning dashboard developed with Python, Scikit-learn and Streamlit.

## Features

- PCA visualization
- Logistic Regression
- LDA
- QDA
- Decision Tree
- KNN
- One-Class SVM
- Confusion Matrix
- Classification Report
- Random image testing
- Image upload
- Mathematical explanation

## Run the app

```bash
streamlit run app.py
